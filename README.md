# themesender
- Web app with Firebase Auth, FireStore DB, ParcelJS for bundling and ReactJs as UI

## Starting up
```
 git clone https://github.com/michaelperryjnr/themsender.git && cd themesender && npm install
```
## Firebse setup..
- Create a Firebase project, add a web app..
- Copy your firebase config from your project settings.
- Enable FireStore, FireStore DB, Authentication(Email and Password Auth and Google Auth) and Storage
